# What?
Find '+' char and replace char before to Upper. And same for '-' char.

# Before...
* g++ 4.2.1 or later
* Download ```reChar-master.zip```
* Exctract it

# Let's Play :D
* Open terminal
  * ```g++ rechar.cpp -o rechar```
  * ```./rechar```
  * Input text, for example: ```R-ec+H-ar```
