# What?
Find '+' and change before to Upper. And same for '-' change to Lower.

# Before...
* g++ 4.2.1 or later
* Download [reChar-master.zip](https://github.com/HilmiZul/reChar/archive/master.zip)
* Exctract it

# Let's Play :D
* Open terminal
  * ```g++ rechar.cpp -o rechar```
  * ```./rechar```
  * Input text, for example: ```R-ec+H-ar```
